Расчет укрупненных показателей потребления тепла жилого здания

Вводите значения только в желтые поля. Результаты расчета автоматически отображаются в таблице.

Исходные данные

Название объекта строительства
Город строительства
Температура холодной пятидневки, °С

Можно вводить название с клавиатуры: при совпадении с городом из списка температура подставится автоматически.

Потребители (здания/секции)

Горячее водоснабжение (ГВС) общее на объект

Способ задания
Расход воды на ГВС по объекту, м³/ч
Нагрузка на ГВС по объекту, кВт
