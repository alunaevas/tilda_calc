Расчет воздухообмена автостоянки по ОНТП-01-91

Вводите значения только в желтые поля. Все расчетные данные ниже обновляются автоматически.

Исходные данные

Количество машиномест на стоянке, шт
Процент въездов в час, % (по умолчанию 2% согласно СП60)
Процент выездов в час, % (по умолчанию 8% согласно СП60)
ПДК CO в рабочей зоне, мг/м³
ПДК CH в рабочей зоне, мг/м³
ПДК NOx в рабочей зоне, мг/м³
Концентрация CO в приточном воздухе, мг/м³
Кол-во газов, проникающих через местный отсос, % (если МО нет, то 100%)
Тип автомобиля
Наименование сооружения стоянки
Режим движения автомобиля

Расчетные данные

Удельный выброс CO одним автомобилем q, г/км
Удельный выброс CH одним автомобилем q, г/км
Удельный выброс NOx одним автомобилем q, г/км
Условный пробег одного автомобиля въезд L, км
Условный пробег одного автомобиля выезд L, км
Коэффициент режима движения для CO
Коэффициент режима движения для CH
Коэффициент режима движения для NOx
Количество CO, выделяемых в атмосферу М, г/с
Количество CH, выделяемых в атмосферу М, г/с
Количество NOx, выделяемых в атмосферу М, г/с
Объемный расход удаляемого воздуха по CO L, м³/ч
Объемный расход удаляемого воздуха по CH L, м³/ч
Объемный расход удаляемого воздуха по NOx L, м³/ч
Результирующий воздухообмен L, м³/ч

Расчет выполнен по ОНТП-01-91 п. 3.2.
